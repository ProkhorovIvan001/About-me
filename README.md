# Всё обо мне.

Я верю в первую очередь в себя, а также в ритм — и в музыке, и в жизни.

Мои хобби - клавиши фортепиано и беговые дорожки, вокал и джиу-джитсу — для меня это разные грани одного целого: дисциплины, выразительности и движения вперёд. 

А ещё в планах — завести сиба-ину, чтобы разделять с ним это динамичное путешествие.
